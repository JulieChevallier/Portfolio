Projects that I've been worked

Double Snake

Double Snake game made in C++ with the SDL 2 library in November 2022.

The double snake is a game inspired by the famous snake game.
But this time, the objective is to guide his own snake to kill the opponent by leading him to the collision. To do this, food is available on the screen to make your snake grow. Be careful, to win you will have to avoid collisions with walls, the body of the opponent's snake as well as your own body.
The arrow keys change the direction of the snake for player 1 and zqsd for player 2. RETURN (ENTER) pauses the game.

Github/Double-Snake

Puzzle Solving Algorithm

Made in Java, this project was programming n Intellij IDEA IDE in April 2022.

This algorithm allows to solve by total exploration any "puzzle".
To improve the efficiency of the algorithm, it uses Collections in Java which allows it to solve any puzzle in a short time.
This project contains three different examples of puzzles that can be solved with the algorithm, the Taquins, the Towers of Hanoi and the famous Sudoku.

Github/Puzzle-Solving-Algorithm

Votee

The site Votee made in HTML/CSS, JavaScript, PHP and SQL on PhpStorm IDE in December 2022.

The Votee application is a web application for collective decision-making (or participatory democracy). Once deployed on a permanently open server, this web application allows to manage a large set of questions (independent of each other).
